# YouTube without Spoilers

[English](README.md) · **Русский**

Chrome-расширение, которое скрывает полосу прогресса, длительность видео, главы и длительности на превью YouTube — **по конкретным каналам**. Смотри турниры и спортивные нарезки с сохранённой интригой.

> В Chrome Web Store: _(ссылка появится после ревью)_

## Зачем

Включаешь нарезку с турнира. Кто победил — не знаешь. Бросаешь взгляд на полосу прогресса — осталось 5 минут, твоя команда проигрывает 2-3. Интрига сломана.

Большинство существующих «спойлер-блокеров» работают по принципу «всё или ничего» или вообще отключены Chrome за то что не мигрировали на Manifest V3. Это расширение сразу спроектировано вокруг идеи whitelist-а по каналам.

## Что умеет

- Скрывает **полосу прогресса** в плеере (клики по ней работают — можно перематывать вслепую, это даже усиливает азарт)
- Скрывает **текущее время** и **общую длительность**
- Скрывает **превью кадра** при наведении на таймлайн
- Скрывает **главы** в заголовке и chapter-маркеры на полосе
- Скрывает **бейджи длительности** («12:34») на превью видео в ленте, поиске и на странице канала
- Скрывает **автоплей превью** при наведении на карточку
- **Whitelist по каналам** — только те каналы, которые ты выбрал. Или режим «везде на YouTube» одним кликом.
- Мастер вкл/выкл из popup-а или по `Alt+H`
- Каждый элемент отключается отдельно

## Установка

### Из Chrome Web Store
_(ожидает ревью)_

### Вручную (developer mode)

1. Скачай или клонируй этот репозиторий
2. Сгенерь PNG-иконки:
   - Открой `icons/generate.html` в Chrome → кнопка «Скачать все 3 PNG»
   - Перенеси скачанные `icon16.png`, `icon48.png`, `icon128.png` в `icons/`
   - _(PNG-иконки уже лежат в репо, этот шаг нужен только если ты редактировал `icon.svg`)_
3. Открой `chrome://extensions/` → включи **Режим разработчика** (правый верх)
4. **Загрузить распакованное** → выбери эту папку
5. Закрепи расширение в тулбаре

## Как пользоваться

1. Зайди на YouTube-канал где хочешь скрывать спойлеры
2. Кликни иконку расширения → **+ Добавить**
3. Готово. Все видео с этого канала будут зацензурены — и в плеере, и в фиде.

Чтобы убрать канал — открой popup → жми `×` рядом с названием.

## Приватность

Никакой аналитики, телеметрии, сторонних сервисов. Настройки хранятся только в `chrome.storage.sync`.
