# YouTube without Spoilers

[English](README.md) · **Русский**

Chrome-расширение которое скрывает на YouTube таймлайн, длительность видео, главы и длительности на превью — **по каналам**. Смотри турниры и спортивные нарезки с полностью сохранённой интригой.

> В Chrome Web Store: _(ссылка появится после ревью)_

## Зачем

Включаешь нарезку с турнира. Кто победил — не знаешь. Бросаешь взгляд на полосу прогресса — осталось 5 минут, твоя команда проигрывает 2-3. Интрига сломана.

Большинство похожих расширений работают по принципу «всё или ничего» либо вообще отключены Chrome за то что не мигрировали на Manifest V3. Это — Manifest V3, поддерживается, и спроектировано вокруг идеи whitelist-а по каналам.

## Что скрывает

- **Полоса прогресса** в плеере (остаётся кликабельной — можно перематывать вслепую для дополнительного азарта)
- **Текущее время** и **общая длительность**
- **Превью кадра** при наведении на таймлайн
- **Главы** в названии и chapter-маркеры на прогресс-баре
- **Длительность** на превью видео в ленте, поиске и на странице канала
- **Автоплей превью** при наведении на карточку
- **Whitelist по каналам** (по умолчанию) либо режим «везде на YouTube»
- Master on/off через popup или `Alt+H`
- Каждый элемент можно отключить отдельно

## Установка

### Из Chrome Web Store
_(в процессе ревью)_

### Вручную (developer mode)

1. Скачай или склонируй репо
2. Сгенерь PNG-иконки:
   - Открой `icons/generate.html` в Chrome → кнопка «Скачать все 3 PNG»
   - Перенеси скачанные `icon16.png`, `icon48.png`, `icon128.png` в `icons/`
3. Открой `chrome://extensions/` → включи **Режим разработчика** (правый верхний угол)
4. **Загрузить распакованное** → выбери эту папку
5. Закрепи расширение в тулбаре

## Как пользоваться

1. Зайди на канал YouTube где хочешь скрыть спойлеры
2. Кликни иконку расширения → **+ Добавить**
3. Готово. Все видео этого канала будут зацензурены — и в плеере, и в фиде.

Чтобы убрать канал: открой popup → жми `×` рядом с его именем.

## Приватность

Никакой аналитики, телеметрии, сторонних сервисов. Настройки хранятся только в `chrome.storage.sync`.
