Генератор иконок No Spoilers

Открой этот файл в Chrome file://.../icons/generate.html, нажми кнопку — браузер скачает три PNG. Положи их в эту же папку icons/ с именами icon16.png, icon48.png, icon128.png.

  1. Жми кнопку
  2. Chrome скачает 3 файла в ~/Downloads
  3. Перенеси их в папку icons/ рядом с этим HTML
  4. Reload расширения в chrome://extensions/ → иконка появится в тулбаре
